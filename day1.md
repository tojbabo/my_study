# 2일차 - Git

JDK란 Java Developement Kit

C 아래 Java bin아래에 관련 키트가 자동으로 설치된다.

주요 키트는 java, javac
java - 실행 명령어, 인터프리터
javac - (compiler) 컴파일 명령어, 컴파일러 -> 이 두 개가 주요 명령어다.


Java의 개발 환경
1. Standard(표준) - SE(Standard Edition)
	: 자바 기본, JDK를 통해 구축

2. Enterprise(기업) - EE(Enterprise Edition)
	: web개발

3. Mobile(micro) - M(Micro Edition)

Java의 압축파일 형식 3가지
jar, war, ear
jar ( java archive ) : .class 파일들의 압축파일
war ( web archive ) : web application 압축파일
ear ( enterprise archive ) : 

.java(소스파일) -> javac -> 
.class(컴파일된 파일) -> java -> 실행

JRE(Java Runtime Environment)
hw-os-jvm-.class 까지의 환경을 구축하는 것

그리고 javac, javadoc, jar등의 명령어(jdk)를 통해 실행
